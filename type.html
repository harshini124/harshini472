Typography Rules

This document defines the typography rules for our website using the fonts Playfair Display and Inter.

Headings

H1 - Main Title: Font: Playfair Display, Weight: 700, Size: 36px, Line Height: 48px

H2 - Sub Title: Font: Playfair Display, Weight: 500, Size: 28px, Line Height: 38px

Body Text

Body: Font: Inter, Weight: 400, Size: 16px, Line Height: 24px

Usage

Headings are used for titles and sections. Body text is used for paragraphs and general content.
